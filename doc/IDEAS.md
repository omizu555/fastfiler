# FastFiler 機能追加・改善 提案

現在の v3.0 (ドック式ペイン配置) 完成時点で、追加すると面白そう・便利そうなアイデアを並べました。
**実装難易度** ☆〜☆☆☆☆☆ で目安、**効果** はユーザ体験への寄与度 (5 段階)。

> 「やる / やらない / 後で」を ★ で印を付けてください。提案ベースなので採否はご自由に。

---

## 🔥 即効性が高そうなもの

### 1. クイックアクセス (お気に入り) サイドバー
- ☆☆ / 効果 ★★★★★
- ワークスペースツリーの上に「📌 ピン留め」セクションを追加し、よく使うフォルダを 1 クリックで開ける。
- D&D でツリーやファイルリストから追加 / 右クリックで並べ替え・削除。
- 既存の `localStorage` (`fastfiler:state:v1`) に `pinned: { name, path }[]` を追加するだけ。

### 2. アドレスバー (パンくずを編集モードに)
- ☆☆ / 効果 ★★★★☆
- パンくずをクリックすると一行テキストに切替、パスを直接打ち込めるように。
- `Ctrl+L` で即座に編集モード (Chrome / Explorer 同等)。
- パス補完 (Tab) も実装すれば爽快。

### 3. ファイル名インクリメンタルサーチ
- ☆ / 効果 ★★★★☆
- ペインにフォーカスしてキーを叩くと、先頭一致でハイライトしてジャンプ。
- 一定時間 (300ms) 入力が無ければバッファクリア。
- 既存の Ctrl+F (再帰検索) とは別物として併用。

### 4. ファイル操作の Undo
- ☆☆☆ / 効果 ★★★★★
- リネーム / 移動 / コピー / ゴミ箱送り を直前 N 件まで `Ctrl+Z` で取り消せる。
- 履歴は in-memory (再起動でクリア) でも十分実用的。

### 5. プログレスダイアログ (大量コピー / 移動)
- ☆☆☆ / 効果 ★★★★☆
- 現在は同期的にブロックしているはずなので、進捗 UI と「キャンセル / バックグラウンド化」を追加。
- Tauri 側で `tauri::Window::emit` を進捗イベントとして発行 → フロントでトースト的に表示。

## 🎨 UI / UX 改善

### 6. テーマカスタマイズ
- ☆☆ / 効果 ★★★☆☆
- 設定に「アクセントカラー」のカラーピッカーを追加。CSS 変数 `--accent` を上書き。
- ダーク / ライトの中間 (高コントラスト / セピアなど) も用意。

### 7. ステータスバー
- ☆ / 効果 ★★★☆☆
- ペイン下部に「選択 N 件 / 合計サイズ / 現在ドライブ空き容量」を表示。
- 最近のエクスプローラと同等の安心感。

### 8. キーボードチートシート (Ctrl+/ で表示)
- ☆ / 効果 ★★☆☆☆
- 既存の `state.hotkeys` を一覧表示するモーダル。
- 設定を開かずに「あれ何だっけ？」を即解消。

### 9. ファイルアイコンのカスタマイズ (拡張子ごと)
- ☆☆ / 効果 ★★★☆☆
- 既定では絵文字だが、ユーザがアイコンセット (Devicons / VSCode Icons) を選べる。
- SVG を内蔵して切替するだけ。

### 10. パネルプリセット
- ☆☆ / 効果 ★★★★☆
- 「コーディング用 / 画像確認用 / バックアップ用」など、レイアウト + 開いているタブをまとめて保存・呼出。
- 設定ダイアログに「プリセット管理」タブ。

## 🧠 ちょっと尖った機能

### 11. デュアルパネル「同期スクロール」
- ☆☆ / 効果 ★★★☆☆
- 連動 Red/Blue に「scroll 同期」をすでに ON/OFF できるが、**列単位** (位置 % で同期) も選べるようにする。
- フォルダ比較 (差分確認) が劇的に楽。

### 12. フォルダ比較モード
- ☆☆☆☆ / 効果 ★★★★☆
- 2 ペインのファイル一覧を並べ、**両方にある / 片方のみ / サイズ違い** で色分け。
- バックアップ確認や写真整理で活躍。
- 比較は名前 (+サイズ / +更新日時) のオプションで。

### 13. タグ / カラーマーク
- ☆☆☆ / 効果 ★★★☆☆
- ファイルに自分用のラベル (赤・青・★) を付ける。実体は変えず、`localStorage` または隠しサイドカーファイル (`.fastfiler-tags.json`) に保存。
- フィルタや並び替えに利用。

### 14. ファイル「クイックビュー」(Spacebar)
- ☆☆☆ / 効果 ★★★★☆
- macOS の Quick Look 相当。スペースキーで現在の選択をフルスクリーン プレビュー (画像 / テキスト / PDF)。
- 既存の PreviewPane を全画面モーダル化するイメージ。

### 15. 「最近開いたフォルダ」リスト + Ctrl+Shift+T
- ☆ / 効果 ★★★☆☆
- 閉じたタブを `Ctrl+Shift+T` で復活 (ブラウザ風)。
- アドレスバーのドロップダウンで履歴表示。

### 16. ターミナルパネル (PowerShell / cmd 起動)
- ☆☆☆☆ / 効果 ★★★★★ (開発者向け)
- ペインのカレントフォルダで PowerShell を開く dock パネル。
- 既存の dock 機構に乗せるだけで自然に統合可能。
- xterm.js + Tauri の `Command::new` で実装可。

### 17. ドラッグ中のフォルダ展開 (Spring-loaded folder)
- ☆☆ / 効果 ★★★☆☆
- ファイルを掴んでツリーのフォルダ上に **0.7 秒ホバー** すると自動展開、その先のフォルダにドロップできる。
- Mac Finder / Windows 11 のあれ。

### 18. プラグインマーケット (ローカル)
- ☆☆☆☆ / 効果 ★★★☆☆
- `%APPDATA%\fastfiler\plugins\` に手動で置く現状を、設定ダイアログから一覧 / インストール / 有効化 / 削除できる UI に。
- ZIP インポート対応。

## 🛠 中身の改善 (技術寄り)

### 19. USN ジャーナル監視 (NTFS)
- ☆☆☆☆☆ / 効果 ★★★★★
- 現在 `notify` クレート (ReadDirectoryChangesW) でフォルダ単位監視 → USN なら全ボリュームのファイル変更を瞬時に拾える。
- Everything 連携と組み合わせれば検索インデックスもリアルタイム反映可能。

### 20. ネイティブ IContextMenu 統合
- ☆☆☆☆☆ / 効果 ★★★★★
- 右クリックを Windows のシェル拡張メニューと統合 (圧縮ツール / Git GUI / クラウド同期等が直接呼べる)。
- 既存の plan.md でも将来候補にあるやつ。

### 21. OLE D&D 完全対応
- ☆☆☆☆ / 効果 ★★★★☆
- 他アプリ (Outlook / ブラウザ / 画像ビューア) からのドロップ、他アプリへのドロップ。
- HTML5 D&D だけだとファイルパスが取れない／取りにくい場面がある。

### 22. パフォーマンス計測パネル
- ☆☆ / 効果 ★★☆☆☆ (開発者向け)
- 設定 → 「デバッグ」タブで FPS / 列挙時間 / サムネイル生成時間などを表示。
- パフォーマンス回帰の早期発見に。

### 23. ファイルサイズの単位カスタマイズ
- ☆ / 効果 ★★☆☆☆
- 1024 (KiB) と 1000 (KB) 切替 / 常に MB 表示 / 自動。

## おすすめ優先度 (個人の独断)

1. **クイックアクセス** (#1) — 毎日効く
2. **アドレスバー編集** (#2) — Ctrl+L で爆速移動
3. **インクリメンタルサーチ** (#3) — 標準エクスプローラ越え
4. **Undo** (#4) — 安心感が段違い
5. **クイックビュー Spacebar** (#14) — 写真整理が捗る
6. **ターミナルパネル** (#16) — 開発フローに直結
7. **フォルダ比較** (#12) — 一度味わうと戻れない

逆に、ネイティブ IContextMenu (#20) や USN (#19) は効果は大きいが Rust + Win32 の沼なので、機能要望が出てから着手で十分かも。
